Import / Export Settings

Use this section to quickly import and export QMAL settings, so that you can use the same settings on different devices.

Since QMAL is not available in the Chrome Web Store, settings do not get saved when you update QMAL to a newer version. Using this section, you can save your settings locally in your system to import when you update QMAL.

Encrypting your settings with a passphrase is probably a good idea, because your MAL username and password also gets exported, and can easily be opened and seen by anyone who has access to the setting file. By encrypting it with a passphrase that only you know, you can safely import it from another device without anyone knowing your credentials on the way.
